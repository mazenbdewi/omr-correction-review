# OMR Correction Landing Page

موقع هبوط عربي RTL لشرح وتسويق برنامج تصحيح الأوراق الامتحانية OMR.

## الملفات
- index.html
- styles.css
- script.js

## طريقة التشغيل
افتح ملف `index.html` مباشرة في المتصفح.

## ملاحظات
- لا يحتوي الموقع على أسعار.
- لا يحتوي على نموذج تواصل.
- يمكن لاحقاً إضافة زر واتساب أو نموذج طلب عرض حسب طريقة البيع.
- التصميم يعمل بدون أي مكتبات خارجية.
